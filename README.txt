How to compile SimCity program:
1. On a Linux machine, go to the "SimCity" directory.
2. Run the "make" command.
3. Four files, including a "SimCity" file will be created in the directory.


How to run SimCity program:
1. Type "./SimCity" to start the program.
2. Enter the name of the configuration file.
3. The program will automatically exit after the initial region state is displayed.
